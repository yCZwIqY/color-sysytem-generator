## 🔹 소개
**Color System Generator**는 **HEX 색상 코드**를 기반으로 **유사한 색상을 자동 생성**하는 도구입니다.  
디자이너, 개발자, 브랜드 기획자 등 다양한 분야에서 활용할 수 있습니다.

## ✨ 주요 기능
✅ **HEX 기반 유사색 생성**  
✅ **100 ~ 900 단계의 색상 스케일 지원**  
✅ **원클릭 HEX 값 복사 기능**<br/>
✅ **빠르고 직관적인 UI**
